# Dynamic Dark Portfolio Website

A modern, responsive single-page portfolio website featuring a sleek dark theme with dynamic animations and interactive elements.

## Overview

This portfolio showcases your work, skills, and experience through a beautifully designed dark-themed interface. Built with pure HTML, CSS, and JavaScript—no frameworks or build tools required.

## Features

- **Dark Theme**: Elegant dark color palette with vibrant orange and purple accents
- **Fully Responsive**: Optimized for phones, tablets, laptops, and large desktops
- **Dynamic Animations**: Scroll-reveal effects, hover transitions, and subtle micro-interactions
- **Interactive Elements**:
  - Animated hero section with SVG graphics
  - Project filtering with smooth transitions
  - Contact form with real-time validation
  - Mobile hamburger navigation
  - Smooth scrolling with active nav highlighting
- **Data-Driven**: All content managed through a single configuration file

## Tech Stack

- HTML5 (semantic markup)
- CSS3 (custom properties, flexbox, grid, animations)
- Vanilla JavaScript (ES6+)

## How to Run

Simply open `index.html` in any modern web browser. No server or build process required.

## Customization

Edit `js/config.js` to personalize your portfolio:

1. **Owner Info**: Update your name, role, tagline, and bio
2. **Contact**: Add your email and social media links
3. **Skills**: Organize your technical skills by category
4. **Projects**: Add your projects with descriptions, tech stack, and links
5. **Experience**: Document your work history and achievements

Each section in the config file includes helpful comments to guide you.
